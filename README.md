## 🧠 Architecture

This project follows a **feature-based architecture** with separation of concerns:

lib/

```
lib/
├── core/
│   ├── routes/
│   ├── theme/
│   ├── utils/
│   └── di/
│
├── features/
│   ├── home/
│   ├── search/
│   ├── product_details/
│   ├── cart/
│   ├── checkout/
│   └── profile/
```

* Feature-first structure
* Modular and reusable widgets
* Scalable routing system
* Dependency Injection ready
